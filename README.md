## How to run this website

1. Clone repository
2. Open $$Cheese-database.Rproj$$
3. Open $$cheese_flexdashboard.Rmd$$
4. Hit $$Run Document$$ to generate flexdashboard
5. In the flexdashboard, click $$Open in Browser$$ for full interactability
